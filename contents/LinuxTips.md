## 常用指令
* 卸载snap软件
``` bash
sudo snap remove xxxx
```

* 给开权限
``` bash
chmod 777 xxxx
```

* 安装.deb包的软件
``` bash
sudo dpkg -i xxxxx
```
* 用vscode打开文件
``` bash
code (xxx/xxx/)xxx
```


* tar压缩与解压缩
``` bash
# 解压
tar -zxvf FileName.tar.gz
# 压缩
tar -zcvf FileName.tar.gz DirName
```
